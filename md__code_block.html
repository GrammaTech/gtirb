CodeBlock

A CodeBlock represents a basic block in the binary.

The corresponding Protobuf message type is CodeBlock.

Guaranteed Properties

  • A CodeBlock is a ByteBlock and a CfgNode.
  • Each CodeBlock must store its decode_mode, and this value must be readable and writable.
