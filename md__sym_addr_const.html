SymAddrConst

A SymAddrConst represents a symbolic operand of the form "Sym + Offset".

The corresponding Protobuf message type is SymAddrConst.

Guaranteed Properties

  • SymAddrConst is one of the permissible types for a SymbolicExpression.
  • Each SymAddrConst object must have the following information, and the API must provide functionality for getting and setting each.
    • offset
    • symbol_uuid (as a Symbol)
