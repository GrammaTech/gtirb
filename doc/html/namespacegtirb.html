Detailed Description

GTIRB_USE_ANY_IMPL_HEADER

The user can toggle between boost and std implementation of "any". This is not a feature, but a work-around for older compiler support. Once all compilers support std::any, this should go away. GTIRB_USE_ANY_IMPL_HEADER is defined by the build system (CMake or SCons) to either be <boost/any.hpp> or <any>.

GTIRB_GTIRB_EXPORTS

Defined by the build system (CMake or SCons). This should only be defined by the build system which generates the GT-IRB library. Users of the library should NOT define this. GTIRB_GTIRB_EXPORT_API

This controls the visibility of exported symbols (i.e. classes) in Windows DLL's and Linux Shared Objects.

The namespace for the GT-IRB library.
