Detailed Description

Author
John E. Farrier

An exception indicating a GT-IRB node has an error specific to the node's type. An example would be when validation of its data failed at some critical point.

void gtirb::NodeError::setNodeType ( std::string  x)

Manually set the name of the node's type. Generally, typeid should be used for retrieving the name. Because this is not set automatically, a name demangler could be used prior to calling this function.

auto e = gtirb::NodeError("Demo Error.");
e.setNodeType(typeid(*this).name());
