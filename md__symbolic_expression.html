SymbolicExpression

A SymbolicExpression represents a data value or instruction operand which should be interpreted as referring to a symbol.

Guaranteed Properties

  • A SymbolicExpression must be one of the following, implemented as either an inheritance hierarchy or a tagged union.
    • SymStackConst
    • SymAddrConst
    • SymAddrAddr
