Enumerator
Part0 

Only bytes [0,1] are used.

Part1 

Only bytes [2,3] are used.

Part2 

Only bytes [4,5] are used.

Part3 

Only bytes [6,7] are used.

Adjusted 

Indicates that the Parts are composed by addition, and that the lower part is sign extended. Thus the upper parts must be adjusted to account for that. Applies to Part1-Part3, but not Part0.

GotRef 

Value is the address of the GOT entry for the symbol.

GotRelPC 

Value is the PC-relative address of the GOT entry for the symbol. How the PC relates to the address of the instruction is ISA-dependent.

GotRelGot 

Value is the address of the GOT entry for the symbol relative to the global pointer, which is the address of the GOT plus an offset. The offset is ISA-dependent, and in the case of certain PPC ABIs, may also depend on which object file the instruction originally came from.

AddrRelGot 

Value is the address as computed by the symbolic expression relative to the global pointer, defined above.

GotRelAddr 

Value is the negative of GOT_RELGOT. That is, it is the global pointer relative to the address of the GOT entry for the symbol.

GotPage 

Value is the address of a special GOT entry for the symbol, relative to the global pointer.

GotPageOfst 

Value is the address of the symbol, relative to the GOT_PAGE value.

PltRef 

Value is the address of the PLT entry for the symbol.
