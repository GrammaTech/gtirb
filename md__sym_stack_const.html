SymStackConst

A SymStackConst represents a symbolic operand of the form "Sym + Offset", representing an offset from a stack variable.

The corresponding Protobuf message type is SymStackConst.

Guaranteed Properties

  • SymStackConst is one of the permissible types for a SymbolicExpression.
  • Each SymStackConst object must have the following information, and the API must provide functionality for getting and setting each.
    • offset
    • symbol_uuid (as a Symbol)
