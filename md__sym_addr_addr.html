SymAddrAddr

A SymAddrAddr represents a symbolic operand of the form "(Sym1 - Sym2) / Scale + Offset".

The corresponding Protobuf message type is SymAddrAddr.

Guaranteed Properties

  • SymAddrAddr is one of the permissible types for a SymbolicExpression.
  • Each SymAddrAddr object must have the following information, and the API must provide functionality for getting and setting each.
    • scale
    • offset
    • symbol1_uuid (as a Symbol)
    • symbol2_uuid (as a Symbol)
